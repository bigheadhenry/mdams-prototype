# 常见问题与排障

如果你在使用 MDAMS 时遇到问题，优先按下面顺序排查：

1. 阅读 [`SETUP_AND_DEPLOYMENT.md`](SETUP_AND_DEPLOYMENT.md)
2. 查看 [`ACCEPTANCE_CHECKLIST.md`](ACCEPTANCE_CHECKLIST.md)
3. 检查 [`WORK_LOG.md`](WORK_LOG.md)
4. 确认 `.env` 是否按当前环境配置正确

当前最常见的问题包括：

- 前端打不开
- 后端健康检查不通过
- Cantaloupe 无法启动
- Mirador 无法加载图像
- 上传目录未挂载
- 数据库连接失败

如果仍然不能解决，建议继续查看：

- [`PROJECT_STATUS.md`](PROJECT_STATUS.md)
- [`AUTH_AND_IIIF_INTEGRATION_PLAN.md`](AUTH_AND_IIIF_INTEGRATION_PLAN.md)

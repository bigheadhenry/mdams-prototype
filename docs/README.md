# 文档索引

当前项目正式文档统一放在 `docs/`。

这轮文档治理基线见：

- `01-总览/DOCUMENTATION_UPDATE_PLAN.md`

## 目录结构

- `01-总览/`
  项目状态、计划、验收、测试策略、工作日志、文档更新方案
- `02-架构设计/`
  系统架构、平台适配器、API 路由、认证与 IIIF、三维子系统
- `03-产品与流程/`
  角色权限、菜单矩阵、工作流、图像记录工作台说明
- `04-实施方案/`
  各阶段实施基线与归档方案
- `05-部署与运维/`
  部署、环境变量、排障、脚本与批处理说明
- `06-参考资料/`
  元数据参考、样例、参考资源导入说明
- `07-图示/`
  架构图与图源文件
- `08-研究/`
  研究子项目材料
